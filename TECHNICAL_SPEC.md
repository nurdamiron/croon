# Техническое задание: Копия сайта croon.kz

## 1. Цель проекта

Создать полную независимую копию интернет-магазина [croon.kz](https://croon.kz/), работающую без зависимостей от платформы InSales. Сайт должен быть визуально идентичен оригиналу и обладать полной функциональностью e-commerce.

---

## 2. Стек технологий

| Компонент      | Технология                        |
| -------------- | --------------------------------- |
| Frontend       | Next.js 14 (App Router) + Tailwind CSS |
| Backend/API    | Next.js API Routes                |
| База данных    | PostgreSQL                        |
| ORM            | Prisma                            |
| Аутентификация | NextAuth.js                       |
| Админ-панель   | Встроенная (Next.js)              |
| Хостинг        | PS.kz (Казахстан)                 |
| Оплата         | Будет определена позже            |

## 6. Модель базы данных (PostgreSQL)

```
User
  - id, email, password_hash, name, phone, role (USER/ADMIN), created_at

Category
  - id, name, slug, parent_id (self-ref), image_url, sort_order

Product
  - id, name, slug, description (HTML), price, sale_price,
    in_stock, category_id, created_at, updated_at

ProductImage
  - id, product_id, url, alt, sort_order

Order
  - id, user_id, status, total, name, phone, email,
    address, delivery_method, payment_method, created_at

OrderItem
  - id, order_id, product_id, quantity, price

Favorite
  - id, user_id, product_id

Page
  - id, title, slug, content (HTML)

BlogPost
  - id, title, slug, content (HTML), blog_slug, created_at
```

## 10. Важные требования

1. **Визуальная идентичность** — сайт должен выглядеть один в один как оригинал
2. **Независимость от InSales** — никаких ссылок, CDN, скриптов от InSales
3. **Все изображения** — скачиваются и хранятся локально
4. **Все URL-ы сохраняются** — для SEO и бесшовного перехода
5. **Валюта** — тенге (тг)
6. **Язык** — русский
7. **Мобильная версия** — полностью адаптивный дизайн

---

## 11. Актуальная эксплуатация (справочно, 2026)

Исходное ТЗ предполагало хостинг PS.kz; **фактически** прод крутится на **AWS** (EC2, RDS, S3). Ежедневный **Google Indexing API** (cron на EC2), деплой через **tar/S3** или GitHub Actions, строки подключения к БД и список workflow — в **`STATUS.md`** и **`CLAUDE.md`** в корне репозитория (эти файлы поддерживаются в актуальном виде).
